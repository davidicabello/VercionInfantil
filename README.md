# Invitación Digital de Cumpleaños

Una invitación digital moderna y elegante para eventos de cumpleaños, construida con HTML, CSS y JavaScript.

## Características

- 🎨 Diseño moderno y responsive
- ⏰ Temporizador de cuenta regresiva
- 📅 Enlaces de calendario automáticos
- 📱 Optimizado para móviles
- 🎭 Animaciones suaves
- 📸 Galería de fotos
- 💬 Integración con WhatsApp

## Tecnologías Utilizadas

- HTML5
- CSS3 (con Bootstrap 5)
- JavaScript (ES6+)
- AOS (Animate On Scroll)
- Google Fonts

## Instalación Local

1. Clona el repositorio:
```bash
git clone https://github.com/tu-usuario/tu-repositorio.git
cd tu-repositorio
```

2. Instala las dependencias:
```bash
npm install
```

3. Ejecuta el servidor local:
```bash
npm start
```

4. Abre tu navegador en `http://localhost:3000`

## Personalización

### Cambiar el contenido
- Edita `index.html` para modificar textos y estructura
- Modifica `styles.css` para cambiar colores y estilos
- Actualiza `main.js` para cambiar la lógica del temporizador

### Cambiar las imágenes
- Reemplaza las imágenes en la carpeta `assets/`
- Actualiza las rutas en `index.html`

### Configurar el evento
- Modifica la fecha, hora y lugar en `main.js`
- Los enlaces de calendario se generarán automáticamente

## Deploy

Este proyecto está configurado para deploy automático en Render. Solo necesitas:

1. Conectar tu repositorio de GitHub a Render
2. El deploy se realizará automáticamente

## Estructura del Proyecto

```
├── index.html          # Página principal
├── styles.css          # Estilos principales
├── main.js            # Lógica JavaScript
├── galeria.html       # Página de galería
├── galeria.css        # Estilos de galería
├── galeria.js         # Lógica de galería
├── sobre.html         # Página "sobre"
├── sobre.css          # Estilos "sobre"
├── assets/            # Imágenes y recursos
├── package.json       # Configuración Node.js
└── render.yaml        # Configuración Render
```
